
# Spotify Clone
This is  a clone website of Spotify build with react js


## Run Locally

Clone the project

```bash
  git clone https://github.com/wearereadytocode/Spotify-clone.git
```

Go to the project directory

```bash
  cd Spotify-clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

  
## Tech Stack

**Client:** React

**Server:** Firebase
